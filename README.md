# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 + JavaScript 实现的经典坦克大战游戏，无需任何外部依赖，打开浏览器即可游玩！

## 🚀 在线试玩

直接在浏览器中打开 `index.html` 即可开始游戏。

## 🎯 游戏特点

- **经典玩法复刻**：还原经典坦克大战的核心玩法
- **多种关卡**：包含3个精心设计的关卡，难度递增
- **多种敌人类型**：
  - 🔴 红色 - 普通坦克
  - 🟣 紫色 - 快速坦克
  - 🟠 橙色 - 重装坦克
- **精美的视觉效果**：爆炸特效、粒子效果、光影渲染
- **完整的UI系统**：开始界面、暂停、游戏结束、过关界面

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `空格` / `J` | 射击 |
| `P` | 暂停/继续 |

## 🏆 游戏目标

- 消灭所有敌方坦克（每关有固定数量）
- 保护基地（底部的 ★）不被摧毁
- 生命耗尽或基地被毁则游戏结束

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript (ES6+)
- 无外部依赖
